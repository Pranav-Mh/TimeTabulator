# TimeTabulator

Full-stack Timetable Management System with MongoDB Atlas.

## Quick start

1. **Start backend:**
   - cd backend
   - npm install
   - cd src
   - node server.js

2. **Start frontend:**
   - cd frontend  
   - npm install
   - npm start

## Features
- ✅ MongoDB Atlas Cloud Database
- ✅ Syllabus Configuration
- ✅ Team Collaboration
- ✅ Real-time Data Sync
